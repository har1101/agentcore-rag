# 山梨県ガイド: 名産品と信玄公祭り

## 1. 山梨県の名産品・特産品

山梨県は四方を山々に囲まれた盆地地形で、寒暖差が大きく晴天率が高いことから「フルーツ王国」として知られる。

### 1.1 フルーツ（果物）

| 果物 | 特徴 | 備考 |
|------|------|------|
| 桃（もも） | 山梨県はもも生産量日本一 | 旬は6月〜9月 |
| シャインマスカット | 酸味が少なく濃厚な甘さ、マスカット香の芳醇な香り | 皮ごと食べられる |
| 巨峰 | 黒ぶどうの代表品種、大粒で濃厚な甘み | 旬は8月〜10月 |
| ぶどう全般 | 山梨県はぶどう生産量日本一 | 甲州ぶどうはワイン原料としても有名 |

### 1.2 郷土料理・グルメ

- **ほうとう**: 山梨県全域で作られる代表的郷土料理。小麦粉を練って作った幅広の麺を生のまま、季節の野菜や肉と一緒に味噌仕立てで煮込んだもの。武田信玄が陣中食として用いたとも伝わる
- **甲府鳥もつ煮**: 鶏の砂肝・ハツ・レバーなどを砂糖と醤油の甘辛いタレで短時間強火で照り煮する甲府市の名物料理。2010年の B-1 グランプリでゴールドグランプリを受賞
- **吉田のうどん**: 富士吉田市の名物。太くてコシの強い麺が特徴で、キャベツと馬肉のトッピングが定番

### 1.3 ブランド肉

- **甲州ワインビーフ**: ワイン醸造後のぶどう粕を飼料に混ぜて育てた牛肉。赤身が美しく旨味が濃い
- **富士桜ポーク**: きめ細やかで柔らかな肉質と適度な霜降りが特徴の高品質ブランド豚肉

### 1.4 銘菓

- **桔梗信玄餅**: 山梨県を代表する銘菓。小さな容器に入った 3 切れのもちにきな粉をまぶし、特製の黒蜜をかけて食べる。桔梗屋が製造
- **ふじさんプリン**: 2026年ジャパンブランドコレクション「日本の贈答品100選」に選出

### 1.5 ワイン

山梨県は日本ワイン発祥の地。甲州市勝沼を中心に 80 以上のワイナリーが集まる。甲州ぶどうを原料とした「甲州ワイン」は国際的にも高い評価を受けている。

### 1.6 伝統工芸品

- **甲州印伝**: 鹿革に漆で模様を付けた革工芸品。印鑑入れ・財布・バッグなどに加工される
- **甲州水晶貴石細工**: 山梨県は宝飾品の産地として国内シェア約 30% を占める

## 2. 信玄公祭り

### 2.1 概要

信玄公祭り（しんげんこうまつり）は、戦国時代の名将・武田信玄の遺徳を偲んで毎年春に山梨県甲府市で開催される祭り。メインイベント「甲州軍団出陣」は **世界最大の武者行列** としてギネス世界記録に認定されている。

### 2.2 第52回（2026年）開催情報

| 項目 | 内容 |
|------|------|
| 開催日 | 2026年4月3日（金）〜 4月5日（日）の 3 日間 |
| 会場 | 甲府駅周辺・舞鶴城公園（甲府城跡）ほか |
| メイン行事 | 甲州軍団出陣（4月4日 土曜日） |

### 2.3 3日間のプログラム

#### 初日（4月3日 金曜日）: 前夜祭
- **湖衣姫コンテスト**: 武田信玄の側室・湖衣姫にふさわしい女性を選ぶコンテスト
- **出陣の宴**: 翌日の出陣に向けた前夜祭イベント

#### 2日目（4月4日 土曜日）: 甲州軍団出陣（メイン）
- 信玄公を中心に武田二十四将をそれぞれ隊将とする軍団が編成される
- **騎馬約30頭、武者約1,600人**の軍勢が舞鶴城公園に集結し出陣
- 武田信玄と上杉謙信が激突した「川中島合戦」への出陣を再現

#### 3日目（4月5日 日曜日）: 賑わい城下町
- 前日の甲州軍団出陣の戦勝を祝う「戦勝を祝う賑わい城下町」イベント
- 地元グルメ・物産展・ステージイベントなど

### 2.4 歴史的背景

武田信玄（1521–1573）は甲斐国（現在の山梨県）の戦国大名。「風林火山」の軍旗で知られ、信濃（長野県）への勢力拡大の過程で越後の上杉謙信と 5 度にわたり川中島で激突した。内政面では信玄堤に代表される治水事業や甲州金の鋳造など、領国経営にも優れた手腕を発揮した。

信玄公祭りは 1970 年（昭和 45 年）に「信玄公まつり」として始まり、以来 50 年以上にわたり甲府市の春の風物詩として親しまれている。
