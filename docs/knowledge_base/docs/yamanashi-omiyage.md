# 山梨のお土産ガイド

## 1. 定番銘菓

### 1.1 桔梗信玄餅（桔梗屋）

山梨県を代表する銘菓。小さな容器に入った 3 切れのもちにきな粉をまぶし、特製の黒蜜をかけて食べる。風呂敷風の個包装が特徴。

| 項目 | 内容 |
|------|------|
| 製造元 | 桔梗屋（1868年創業） |
| 価格 | 6個入（布袋）約1,100円 / 15個入（箱）約3,000円 |
| 受賞歴 | JR東日本おみやげグランプリ金賞（2017年）、モンドセレクション金賞（2012〜2014年3年連続） |
| 由来 | 山梨のお盆に供えるあべ川もちをヒントに1968年発売 |

**バリエーション**:
- **桔梗信玄餅 極（きわみ）**: 容器が食べられるモナカ皮。3個入 約700円
- **桔梗信玄生プリン**: 4個入 1,080円 / 6個入 1,620円
- **桔梗信玄餅 カントリーマアム**: 山梨限定コラボ商品

### 1.2 金精軒の信玄餅

桔梗信玄餅のライバル。1972年発売。黒蜜が溜まるくぼみのある少し小ぶりなもちが特徴。南アルプス（甲斐駒ヶ岳）の天然水と地元「梨北米」を使用。金精軒が「信玄餅」の商標を保有している。

- **極上生信玄餅**: 4個入、賞味期限わずか3日の超フレッシュ版

### 1.3 くろ玉（澤田屋）

1929年から95年以上の歴史を持つ銘菓。うぐいす餡（青えんどう豆の餡）を黒糖羊羹で包んだ丸い球状のお菓子。原材料は上質な青えんどう豆と沖縄県多良間島産の黒糖のみ。

### 1.4 月の雫（つきのしずく）

甲州ぶどうの一粒一粒を手作業で白砂糖の蜜でコーティングした伝統菓子。農林水産省の郷土料理遺産にも認定。秋〜初冬の甲州ぶどうの旬の時期のみ販売される季節限定品。

### 1.5 イタリアンロール（清月）

シュー皮でロールケーキを包んだ独特の食感の洋菓子。モンドセレクション金賞3年連続受賞。プレーン・チョコセレクト・マロンの3フレーバー。プレーン約1,700円。

## 2. ワインのお土産

山梨は日本ワイン発祥の地。甲州市勝沼を中心に80社以上のワイナリーが集まる。日本固有品種「甲州」ぶどうの白ワインは国際的にも高評価。

### 2.1 おすすめワイナリーと銘柄

| ワイナリー | おすすめ銘柄 | 価格（税込目安） | 特徴 |
|-----------|------------|--------------|------|
| グレイスワイン（中央葡萄酒） | Grace Koshu 2023 | 約3,300円 | 1923年創業。Decanter金賞6年連続。日本ワインの世界的評価を確立 |
| サドヤ | シャトー・ブリヤン（赤） | 約3,200円 | 1917年創業。日本における欧州品種栽培のパイオニア |
| 勝沼醸造 | 甲州テロワール・セレクション「祝」 | 約2,300円 | テロワール重視の甲州ワイン |
| シャトー勝沼 | 各種 | 各種 | 149年の歴史。ワイナリーツアー・試飲可能 |

### 2.2 一升瓶ワイン

山梨ならではのお土産：日本酒の一升瓶（1.8L）にワインを詰めた「一升瓶ワイン」。手頃な価格でボリュームがあり、贈り物として人気。

## 3. フルーツ加工品

山梨は「フルーツ王国」。ぶどう・桃・すもも生産量日本一。日照時間の長さと寒暖差が美味しい果物を育てる。

### 3.1 旬のカレンダー

| 果物 | 旬の時期 |
|------|---------|
| さくらんぼ | 6月 |
| 桃 | 6月〜8月 |
| すもも | 6月〜8月 |
| ぶどう / シャインマスカット | 8月〜10月 |

### 3.2 加工品

- **ドライフルーツ**: 桃・ぶどう・キウイ・いちご。山梨県乾燥果実工房なかむらのギフトセットが人気。ドライピーチは芳醇な香りとジューシーな甘さが特徴
- **ジャム**: 地元果物を使った各種ジャム。直売所や道の駅で入手可能
- **果汁のしずく**: 山梨フルーツの果汁を閉じ込めたボンボンキャンディ。シャインマスカット・白桃・黄桃・さくらんぼ・サイダー味
- **フルーツジュース**: 地元果物100%ジュース
- **シャインマスカットポテトチップス**: ネタ系お土産として人気

## 4. 伝統工芸品

山梨県には12の指定伝統工芸品がある。

### 4.1 甲州印伝（こうしゅういんでん）

鹿革に漆で模様を付けた革工芸品。名前の由来はインド（印度）から伝来した装飾革。

- **印傳屋 上原勇七**: 甲州印伝の総本家。江戸時代に初代勇七が技法を確立
- **製品**: 財布・小銭入れ・ポーチ・バッグ・カードケース・ペンケースなど
- **伝統柄**: 鹿・トンボ・菊・幾何学模様
- **本店**: 甲府市中央3-11-15（旧甲州街道沿い）。10:00〜18:00、年中無休（年末年始除く）。2階に印伝博物館（約1,500点収蔵）
- **公式サイト**: https://www.inden-ya.co.jp/

### 4.2 宝石・ジュエリー

山梨は日本のジュエリーの都。宝飾関連企業1,000社以上。水晶の産出をきっかけに貴石細工が発展した歴史がある。

- **甲州貴石細工**: 伝統的な宝石加工技術
- **LUCKY OPEN FACTORY**: 1937年創業。工場見学 + ジュエリー製作体験（5,000円、要予約）。ネックレス/リングの台座5種から選び、自分で宝石をセット・研磨
- **昇仙峡エリア**: ジュエリータウン昇仙峡、クリスタルサウンドなどの宝石施設が集まる

### 4.3 その他の工芸品

- **甲州織**: 富士山の雪解け水で染めた織物。独特の光沢。ネクタイ・バッグ・傘地など
- **甲州雨畑硯**: 硯（すずり）
- **甲州大石紬**: 絹織物
- **西島手漉和紙**: 手漉き和紙
- **親子だるま**: 甲州だるま

## 5. その他の食のお土産

- **ほうとう（贈答用）**: 山梨のソウルフード。生麺タイプのギフトパックで自宅で調理可能
- **あわびの煮貝（みな与）**: 400年以上の歴史を持つ老舗。国産あわびを秘伝のたれで煮込んだ甲州名物。海なし県ならではの保存食文化から生まれた
- **甲州小梅漬**: 小梅生産量日本一。カリカリ食感と酸味・塩味のバランスが特徴
- **甲府鳥もつ煮**: 鶏の砂肝・ハツ・レバーを甘辛いタレで照り煮。2010年 B-1グランプリ ゴールドグランプリ受賞
